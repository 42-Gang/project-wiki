## 🔫 프로젝트 소개
Ping Pong Gang은 탁구 게임을 1:1 게임이나 토너먼트 게임을 실시간으로 즐길 수 있는 곳입니다. 실시간 채팅 기능을 통해 소통하며, Custom 모드를 활용해 원하는 방식으로 토너먼트를 진행할 수 있습니다. 게임 기록을 간편하게 조회하고, 그 기록을 통해 나만의 게임 여정을 되돌아보며, 즐거운 순간들을 쌓을 수 있습니다!
